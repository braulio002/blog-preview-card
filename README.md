# Blog Preview Card

A simple responsive blog preview card built as a practice project for improving my HTML and CSS fundamentals.

## Overview

This project displays a centered card containing a thumbnail, heading, description text, profile picture, and a name.  
The goal was to practice layout structure, spacing, and responsive design principles.

## What I Learned

- The CSS box model (margin, padding, border, box-sizing)
- Centering elements using Flexbox
- Managing spacing with line-height
- Addign immersion with transition and hover
- Importing and applying Google Fonts

## Technologies Used

- HTML5
- CSS3
- DeepSeek

Open `https://braulio002.github.io/blog-preview-card/` in your browser.
